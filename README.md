# MP3 to MIDI Converter - Backend

This is the backend for the MP3 to MIDI Converter application built with Node.js and Express. The backend handles the conversion of MP3 files to MIDI files using `ffmpeg` and `@tonejs/midi`.

## Getting Started

### Prerequisites

- Node.js (version 12 or higher)
- npm (version 6 or higher)
- `ffmpeg` (installed and accessible in your system's PATH)
